# Discord Nuke Bot
* Python 3.6 - 3.11
***
# Installation:
```console
git clone https://github.com/reinxp/Discord-NukeBot.git

cd Discord-NukeBot

python -m pip install -r requirements.txt

Edit config.json

python main.py
```
